# nasm-sb
Exercícios para pré processador e montador NASM
## Como rodar o projeto

### Para rodar switch case:
Dentro do diretório do projeto faça:
`make`
e depois:
`make run`

### Para rodar bcd-arithmetic:
Dentro do diretório do projeto faça:
`cd bcd_arithmetic`
depois:
`make`
e por fim:
`make run`
